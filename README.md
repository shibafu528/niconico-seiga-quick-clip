Niconico Seiga Quick Clip
====

ニコニコ静画のクリップ登録を、クリップ先を選択するだけで済むようにします。  
ニコニコ静画のクリップ登録の操作を簡略化し、クリップ先を選択したらすぐさま登録処理を行うようにします。

複数のクリップを使い分けている方であれば、クリップ先を選び直しては追加ボタン……といった操作を数えられないくらい繰り返していると思います。  
もうそんなことはやめにしましょう。
